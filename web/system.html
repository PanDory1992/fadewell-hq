Granice systemu

Vinted jest źródłem obserwacji ofert: ceny, zdjęcia, likes, views i widoczność. HQ nie wystawia ani nie edytuje ofert na Vinted.

Zdjęcia pozostają URL-ami Vinted — HQ ich nie pobiera i nie przechowuje jako plików.

Stan operacyjny

Supabase / HQ jest jedyną prawdą dla DEN. System pokazuje stan księgi, obserwacje Vinted i sprawy wymagające decyzji, ale nigdy nie zakłada automatycznie sprzedaży.
